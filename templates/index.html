Discovery Brief Agent

Trigger: HubSpot webhook (meeting booked)  ·  Model: Claude Opus 4.6  ·  Repo: discovery-brief-agent

Automatically researches a prospect before a discovery call. When a new meeting is booked in HubSpot, this agent pulls deal and contact data, enriches it via Amplemarket, runs live web research, and generates a 16-section MEDDPICC-style brief saved directly as a HubSpot note on the deal. It also advances the deal stage to "First Meeting Scheduled" so the pipeline stays accurate. The goal is for the AE to walk into every call knowing the prospect's integration landscape, pain points, and ICP fit — without doing any manual research.

Meeting Confirmation Agent

Trigger: Albato automation (3h before call)  ·  Model: Claude Sonnet 4.6  ·  Repo: meeting-confirmation-agent

Sends a personalized meeting confirmation email 3 hours before every discovery call, from the right Albato team member's Front mailbox. For first meetings, it uses a standard intro agenda. For second and further meetings, it fetches recent emails and notes from the HubSpot deal and asks Claude to generate a tailored agenda based on actual next steps and commitments discussed with the lead. The agent creates a Front draft — the AE reviews and sends it manually.

Follow-up Tracker Agent

Trigger: Daily cron 18:00 UTC  ·  Model: Claude Sonnet 4.6  ·  Repo: followup-tracker-agent

Every evening, checks whether the sales team's HubSpot follow-up tasks for the day were actually executed. For each task due today, it gathers evidence — emails sent, calls logged, notes added — and asks Claude to classify the outcome as Completed, Incomplete, Missed, or Not enough data. Posts a structured Slack report to the appropriate channel based on deal stage, giving managers a clear daily view of follow-up discipline across the pipeline.

Pipeline Agent

Trigger: Daily cron 16:00 UTC  ·  Model: Claude Sonnet 4.6  ·  Repo: pipeline-agent

Applies five pipeline rules automatically: advancing deal stages when meetings are completed, fixing incorrect meeting types, extracting Business Fit scores from Sybill call transcripts using Claude, and backfilling contact engagement associations so activity history stays linked to deals. NPS Agent

Trigger: Weekday cron 12:00 + 15:00 UTC  ·  Repo: nps-agent

Automates Albato's quarterly NPS survey end-to-end. Sends surveys via email and Slack, tracks who has responded in a Google Sheet, sends up to three reminder waves to non-respondents, and posts daily response rate updates to Slack. At the end of the quarter, generates a final PDF report with the NPS score, per-client breakdown, and qualitative feedback analysis. Also listens for @Albato NPS Agent mentions in Slack for live Q&A about results.

Action Plan Agent

Trigger: Sybill webhook → Albato → GitHub dispatch  ·  Model: Claude Opus 4.6  ·  Repo: action-plan-agent

Generates a Mutual Action Plan (MAP) immediately after a discovery call ends. Triggered automatically by a Sybill call transcript arriving via Albato. Claude analyzes the transcript to extract concrete next steps, mutual commitments, AE coaching tips, and a nurture vs. push recommendation — then saves the output as a HubSpot note and notifies the sales team on Slack. The goal is to have a structured action plan ready within minutes of the call ending, while context is fresh.
